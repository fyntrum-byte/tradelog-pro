# TradeLog Pro 📊

Journal de trading profesional con calculadora de posición, reglas de cuenta y análisis avanzado.

---

## 🚀 Publicar en Vercel (paso a paso)

### Lo que necesitas
- Computadora con internet
- 10 minutos

---

### PASO 1 — Instala Node.js
Ve a https://nodejs.org y descarga la versión **LTS** (la verde).
Instálala normalmente como cualquier programa.

---

### PASO 2 — Abre la terminal en esta carpeta

**Windows:**
- Abre la carpeta `tradelog-pro`
- Mantén Shift + clic derecho dentro de la carpeta
- Selecciona "Abrir ventana de PowerShell aquí"

**Mac:**
- Abre la carpeta en Finder
- Clic derecho → "Nueva Terminal en la carpeta"

---

### PASO 3 — Instala las dependencias
Escribe esto y presiona Enter:
```
npm install
```
Espera a que termine (puede tomar 1-2 minutos).

---

### PASO 4 — Construye la aplicación
```
npm run build
```
Esto crea una carpeta llamada `dist` con la app lista.

---

### PASO 5 — Sube a Vercel

1. Ve a https://vercel.com y crea cuenta gratis (puedes usar Google)
2. En el dashboard, haz clic en **"Add New → Project"**
3. Selecciona **"Browse"** o **"Upload"**
4. **Arrastra la carpeta `dist`** que se creó en el paso anterior
5. Haz clic en **Deploy**

¡En 30 segundos tendrás tu link! Por ejemplo:
`https://tradelog-pro.vercel.app`

---

### 📱 Para que se vea como app en el celular

**iPhone:**
- Abre el link en Safari
- Toca el botón compartir (□↑)
- Selecciona "Añadir a pantalla de inicio"

**Android:**
- Abre el link en Chrome
- Menú (⋮) → "Añadir a pantalla de inicio"

---

### 🔗 Compartir con tu comunidad
Una vez publicado, simplemente comparte el link de Vercel.
Cada alumno tendrá sus propios datos guardados en su dispositivo.
